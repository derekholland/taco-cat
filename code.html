                    // get the string from the page
                    // Controller Function
                    function getValue() {
                      // add invisible class to alert
                      document.getElementById("alert").classList.add("invisible");
                      let userString = document.getElementById("userString").value;
                      let revString = reverseString(userString);
                      displayString(revString);
                    }
                    //Reverse the string
                    // logic Function
                    function reverseString(userString) {
                      let revString = [];
                      // reverse a string
                      for (let i = userString.length - 1; i >= 0; i--) {
                        revString += userString[i];
                      }
                      return revString;
                    }
                    
                    // display the reversed string to the user
                    // View Function
                    function displayString(revString) {
                      // write message to the page
                      document.getElementById(
                        "msg"
                      ).innerHTML = `Your string reversed is : ${revString}`;
                      // show the alert box
                      document.getElementById("alert").classList.remove("invisible");
                    }

The code in structured into three functions.

getValue() Function

This is the main controller function that calls the other functions in the program. This function is an event the "Reverse Me" button listens for. In the HTML document of this page there is an event listener attached to the "Reverse Me" button. After the user enters a string in the input box and presses the "Reverse Me" button, the getValue()function is called.This functions accepts no arguments. The function starts by adding a CSS class to the div element with an alert id. Adding this class hides the element from the user. Next, the userString variable is created. This variable stores the value of the string the user inputs. Another variable named revString is created. This variable calls the reverseString() function and stores the returned value of that function. Now that we have the reversed String we can pass that to the displayString() function to show the reversed string to the user.

The reverseString() Function.

This function starts by assigning an empty array to the revString variable. This array wil be used to store the reversed string. A for loop runs next to reverse the string. This happens by setting the index of the for loop to the last index of the string in userString. Now the loop will decrement down all the way to zero and store the current character of userString into the revString array on each iteration. This is how the the empty array becomes a copy of the string but in reverse order. The final step of the function returns the reversed string that was just created. Now back in the getValue() function this value is stored in the revString variable.

The displayString() Function

This function displays the reversed string to the user. By using document.getElementById() we can target the element with the "msg" id and set the inner HTML to the message we want to display. Inside of the template literal the variable revString is used to show the final reversed string. To make all of this appear on the page we have to remove the CSS class called invisible. This can be accomplished by targeting the element with the alert class and removing the invisible class from the class list. Now the user can see the reversed string!
